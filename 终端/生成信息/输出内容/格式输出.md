##  格式输出
`\033` 可以使用`\e` 代替
```shell
\033[编号m要改变的内容\033[0m
							|
						恢复默认
```


###   编号
```shell
样色;前景色;背景色
```
```shell
前景色;背景色
```
```shell
前景色
```
```shell
背景色
```
