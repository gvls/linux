##  输入
|-------|---------------------|
| 格式  | 说明                |
|-------|---------------------|
| `0<`  | 标准输入重定向。    |
| `<`   | 清空已有内容。      |
|       | 命令 <==数据== 文件 |
|-------|---------------------|
| `0<<` | 追加输入重定向。    |
| `<<`  | 追加内容到底部。    |
|       | 命令 <==数据== 文件 |
|-------|---------------------|
