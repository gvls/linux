##  输出
|-------|----------------------|
| 格式  | 说明                 |
|-------|----------------------|
| `1>`  | 标准输出重定向。     |
| `>`   | 清空已有内容。       |
|       | 命令 ==数据==> 文件  |
|-------|----------------------|
| `1>>` | 追加输出重定向。     |
| `>>`  | 追加内容到底部。     |
|       | 命令 ==数据==> 文件  |
|-------|----------------------|
| `2>`  | 错误输出重定向。     |
|       | 清空已有内容。       |
|       | 命令 ==数据==> 文件  |
|-------|----------------------|
| `2>>` | 错误追加输出重定向。 |
|       | 追加内容到底部。     |
|       | 命令 ==数据==> 文件  |
|-------|----------------------|
