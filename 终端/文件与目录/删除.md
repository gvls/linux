##  删除
避免误操作，尽量`cd` 到目标目录再使用`rm` 
```shell
rm 参数 文件 文件
```


###   参数
|------|---------------------------------|
| 参数 | 说明                            |
|------|---------------------------------|
| `-f` | 强制删除                        |
|      | 删除不存在的文件/目录也不会提示 |
|------|---------------------------------|
| `-i` | 在删除前 询问                   |
|------|---------------------------------|
| `-r` | 递归删除，可以删除**整个目录**  |
|------|---------------------------------|
