##  新建目录
```shell
mkdir 目录 目录
```

###   参数
|------|---------------------------|
| 参数 | 说明                      |
|------|---------------------------|
| `-p` | 递归创建目录              |
|      | `目录`使用`目录/目录`代替 |
|------|---------------------------|
| `-v` | 显示创建过程              |
|------|---------------------------|
