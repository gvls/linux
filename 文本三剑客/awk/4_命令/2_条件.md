##  条件
也就是 取行

|-----------|------------------------------|
| 方式      | 格式                         |
|-----------|------------------------------|
| 行号/列号 | 使用`NR`/`$列号`             |
|           | 添加`> < >= <= == != && ` 等 |
|           | `NR==开始,NR==结束`          |
|-----------|------------------------------|
| 正则      | `/正则/`                     |
|           | `/开始正则/,/结束正则/`      |
|-----------|------------------------------|
| 包含      | `列~/正则/`                  |
|-----------|------------------------------|
| 不包含    | `列!~/正则/`                 |
|-----------|------------------------------|
