##  BEGIN与END
|---------|---------------------|----------------------------|
| 模式    | 含义                | 使用场景                   |
|---------|---------------------|----------------------------|
| BEGIN{} | 在awk读取文件前执行 | 统计，不涉及读取文件       |
|         |                     | 在处理文件前 添加表头      |
|         |                     | 定义awk变量                |
|---------|---------------------|----------------------------|
| END{}   | 在awk读完文件后执行 | 过程做统计计算，这输出结果 |
|         |                     | 使用数组，这输出数组结果   |
|---------|---------------------|----------------------------|
